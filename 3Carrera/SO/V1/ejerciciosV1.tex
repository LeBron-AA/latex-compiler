\result{Ejercicio 4}
\textbf{Ejecución simultánea de dos programas}

\vspace{2mm}
Programas como el \texttt{programVerySimple} por defecto adjunto a la V1 original, acaban su ejecución con la instrucción \texttt{HALT}. Así, el primer proceso en concluir apaga la simulación al completo, por lo que no puede ejecutarse el otro. Para solucionarlo, hemos de sustituir  la instrucción \texttt{HALT} por \text{TRAP 3} para terminar el proceso de manera ordenada y que el SO pueda ceder el turno de ejecución al siguiente proceso.

\vspace{2mm} \noindent
Ejemplo de salida para dos instancias del siguiente programa:

  ADD 0 -3

  HALT
\end{terminal}
